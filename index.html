📌 計算說明：
內尺寸 = 外尺寸 - 紙層厚度 - 紙板厚度 (長/寬)；內高 = 外高 - 2×(紙層厚度+紙板厚度)
製造尺寸 = 內尺寸 + 紙板厚度 (長/寬)；製造高 = 內高 + 2×紙板厚度
展開長 = (外長+外寬)×2 + 釘接邊；展開寬 = 外寬 + 外高；才數 = (展開長×展開寬)/918
面積 = [(外長+外寬)×2/100 + 0.04] × (外寬/100 + 外高/100 + 0.05) （單位轉為米）
成本 = 面積 × 每平方米單價，顯示於最後一行（綠色高亮）

2. 強度計算 (凱里卡特公式)
P = Px × F
P:抗壓強度(N) Px:綜合環壓(N/cm) F:常數(依楞型周長)
三層：Px = (R1+R2+Rm×C)/15.2
五層：Px = (R1+R2+R3+Rm1×C1+Rm2×C2)/15.2
常數K (F值) 經驗公式
A楞: K=30.3+0.275Z-0.0005Z² C楞: K=27.9+0.265Z-0.0005Z² B楞: K=24.6+0.235Z-0.0005Z²
堆碼強度校核 (GB/T 6543-2008)： P = K × G × (H - h) / h
