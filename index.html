計算說明：
內尺寸 = 外尺寸 − 面紙厚 − 楞高（長/寬）；內高 = 外高 − 2×(面紙厚+楞高)
製造尺寸 = 內尺寸 + 楞高；展開長 = (外長+外寬)×2 + 釘接邊
展開寬 = (外寬+外高)×2（含上下兩組蓋板）；才數 = 展開長×展開寬÷918
面積 = [(外長+外寬)×2÷100+0.04]×(外寬÷100+外高÷100+0.05)；成本 = 面積×單價

1. 尺寸換算公式
類型長/寬方向高度方向參數
製造尺寸內尺寸＋楞高內＋2×楞高A=5 B=3 C=4mm
外尺寸製造＋面紙厚製造＋2×面紙厚三層≈1，五層≈2mm
📌 例：A楞5mm、面紙2mm，內400×260×260mm → 外407×267×274mm
🧮
2. 才數計算 已修正
展開長 = (外長 + 外寬) × 2 + 釘接邊 展開寬 = (外寬 + 外高) × 2 ← 含上下蓋板 才數 = 展開長 × 展開寬 ÷ 918
⚠️ 展開寬需×2，原公式僅用 外寬+外高 為錯誤。
💪
3. 強度計算（凱里卡特公式）
P = Px × F 三層：Px = (R1+R2+Rm×C) / 15.2 五層：Px = (R1+R2+R3+Rm1×C1+Rm2×C2) / 15.2 A楞 K = 30.3 + 0.275Z − 0.0005Z² C楞 K = 27.9 + 0.265Z − 0.0005Z² B楞 K = 24.6 + 0.235Z − 0.0005Z²
堆碼校核 GB/T 6543-2008： P = K × G × (H − h) / h

4. 面積與成本
單位面積公式備註
米(m)[(L+W)×2+0.04]×(W+H+0.05)輸入單位 m
公分(cm)[(L+W)×2+4]×(W+H+5)/10000結果為 ㎡
💰 紙箱價格 = 面積(㎡) × 每平方米單價
