📌 計算說明（已修正）：
紙板總厚度 = 楞高 + 面紙厚度×2（三層）或 楞高×2 + 面紙厚度×3（五層）
內尺寸 = 外尺寸 − 面紙厚度 − 楞高（長/寬）；內高 = 外高 − 2×(面紙厚度 + 楞高)
製造尺寸 = 內尺寸 + 楞高（長/寬）；製造高 = 內高 + 2×楞高
展開長 = (外長+外寬)×2 + 釘接邊；展開寬 = (外寬+外高)×2（上下各一組蓋板）
才數 = 展開長 × 展開寬 ÷ 918；面積 = [(外長+外寬)×2÷100 + 0.04] × (外寬÷100 + 外高÷100 + 0.05)
成本 = 面積 × 每平方米單價

2. 才數計算 已修正
展開長 = (外長 + 外寬) × 2 + 釘接邊
展開寬 = (外寬 + 外高) × 2　← 上下蓋各一組
才數 = 展開長 × 展開寬 ÷ 918
⚠️ 展開寬修正說明：展開後，寬度方向包含前後兩組蓋板（上下各一），因此應乘以2，即 (外寬+外高)×2，而非僅 外寬+外高。
3. 強度計算 (凱里卡特公式)
P = Px × F
P:抗壓強度(N) Px:綜合環壓(N/cm) F:常數(依楞型周長)
三層：Px = (R1+R2+Rm×C)/15.2
五層：Px = (R1+R2+R3+Rm1×C1+Rm2×C2)/15.2
常數K (F值) 經驗公式
A楞: K=30.3+0.275Z-0.0005Z² C楞: K=27.9+0.265Z-0.0005Z² B楞: K=24.6+0.235Z-0.0005Z²
堆碼強度校核 (GB/T 6543-2008)： P = K × G × (H - h) / h
4. 面積與成本
單位面積公式備註
米 (m)[(長+寬)×2 + 0.04] × (寬+高+0.05)長寬高單位為米
公分 (cm)[(長+寬)×2 + 4] × (寬+高+5) / 10000結果為㎡
